Advanced Digging Tutorial

The pivot of a shape can be configured in the inventory ([I]/[Right-Click]) while a digging tool is selected. This controls where the shape is positioned relative to the cursor. Digging with bottom pivot is useful for leveling.
Dig m³ terrain
using the Box shape and Bottom pivot

Advanced Digging Tutorial

The alignment of a shape can be configured in the inventory ([I]/[Right-Click]) while a digging tool is selected. This controls how the shape is rotated and if the position is aligned with respect to a grid.

Advanced Crafting Tutorial

If at least one physics item is placed in the world, a Delete Menu appears in the physics items inventory bag. This menu can delete or collect all dynamic or static items in the world and is intended for emergency removal of physics items that cause performance degeneration.
You may skip this tutorial if you do not want to delete/collect your items
