Basic Recipe Crafting Tutorial

Advanced Digging Tutorial

The pivot of a shape can be configured in the inventory ([I]/[Right-Click]) while a digging tool is selected. This controls where the shape is positioned relative to the cursor. Digging with bottom pivot is useful for leveling.
Hint: This does not work on flat terrain. Find a slope.
Dig m³ terrain
using the Box shape and Bottom pivot

Advanced Digging Tutorial

The alignment of a shape can be configured in the inventory ([I]/[Right-Click]) while a digging tool is selected. This controls how the shape is rotated and if the position is aligned with respect to a grid.
Dig m³ terrain
using a View-aligned Cylinder with Center pivot

Advanced Digging Tutorial

Axis- and grid-alignment depend on a grid. The Y-Angle of this grid can be configured in the inventory after such an aligment was selected.
Dig m³ terrain
using a 45° Grid-aligned Box shape

Advanced Digging Tutorial

Advanced Crafting Tutorial

If at least one physics item is placed in the world, a Deletion Menu appears in the physics items inventory bag. This menu can delete or collect all dynamic or static items in the world and is intended for emergency removal of physics items that cause performance degeneration.
You may skip this tutorial if you do not want to delete/collect your items
Delete or collect all dynamic physics items
